Introduction

"FlexMenu" is a fully responsive jQuery menu/navigation plugin. It was built with Sass CSS (Now also support Less) which makes it easy to modify.

File Structure

Within the download you'll find the following file structure and contents

Once downloaded, unzip the compressed folder and you'll see something like this:

flexmenu/
├── Docs/
├── Example - Basic/
├── Example - Advanced/
└── Flexmenu/
  • "Docs" folder is where the help documentation is.
  • "Example - Baic" contains a simple example project.
  • "Example - Advanced" contains a much complex example project, which also includes Sass CSS and CoffeeScript files.
  • "Flexmenu" contains the minified version of flexmenu plugin.

Basic Usage

You can refer to Example - Basic file for code example. All you need to do is write some HTML, and one line jQuery code.

Include the CSS and JS file

You need jQuery, since it's build on jQuery.

<!-- include the CSS and JS files -->
<link rel="stylesheet" href="flexmenu/css/jquery.flexmenu.css">
<script src="jquery-1.10.2.js"></script>
<script src="flexmenu/js/jquery.flexmenu.js"></script>

The Menu:

<!-- Menu container -->
<div class="fm-container">
	<!-- Button -->
	<div class="fm-button">
		<span class="fm-bar"></span>
		<span class="fm-bar"></span>
		<span class="fm-bar"></span>
	</div>
	<!-- end button -->
	<!-- Menu -->
	<div class="menu">
		<ul>
			<li><a href="#">Item 1</a>
				<ul>
					<li><a href="#">Sub Item 1</a></li>
					<li><a href="#">Sub Item 2</a></li>
				</ul>
			</li>
			<li><a href="#">Item 2</a></li>
			<li><a href="#">Item 3</a></li>
			<li><a href="#">Item 4</a></li>
			<li><a href="#">Item 5</a></li>
		</ul>		
	</div>
	<!-- end Menu -->
</div>
<!-- end Menu container -->

Init, with one line jQuery code:

<!-- init flexmenu -->
<script>
$('.menu').flexMenu();
</script>

Again. You can refer to Example - Basic file for code example.

More options Usage

There are some options avaiable.

  • breakpoint: 767                     # Integer: The responsive breakpoint(px) of flexMenu
  • responsivePattern: 'toggle'   # String: 'toggle' and 'off-canvas'
  • animationSpeed: 250            # Integer: The speed of animations, in milliseconds.

    • So you can write you code this way:

    <script>
$('.menu').flexMenu({
	breakpoint: 800,
	responsivePattern: 'off-canvas',
	animationSpeed: 300
});
</script>
